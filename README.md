\# European high altitude soil microbiome Study


\## Overview

This repository contains R scripts used to analyze soil microbial communities and CH₄/CO₂ fluxes for alpine permafrost-affected sites.



The scripts perform:

\- Data cleaning and processing

\- Statistical analysis

\- Figure generation



\## Data

\- Raw sequences: deposited at ENA under accession \*\*PRJEB109934\*\*

\- Processed data tables and metadata: available on Zenodo: \*\*DOI:10.5281/zenodo.xxxxx\*\*
